# FallLead

**IFA Lead Intelligence Engine**

Sovereign lead qualification tool for Independent Financial Advisers. FCA Consumer Duty compliant. FreudEngine psychology integration. Vulnerability screening.

## Features

- Multi-dimensional lead scoring across financial, psychological, and regulatory dimensions
- FCA Consumer Duty compliance checks built in
- Vulnerability screening (FCA VULN framework)
- FreudEngine psychology layer — unconscious motivator detection
- Bloom vector visualisation per lead
- Qualification verdicts: QUALIFIED / PROMISING / DEVELOPING / RED FLAG
- Risk appetite assessment
- Multi-provider AI: Gemini (free) → DeepSeek (free) → Anthropic (paid)
- All data stays in browser — IndexedDB, sovereign

## Run

Double-click `FallLead.html`. Sovereign. No data leaves your device.
